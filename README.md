# CRUD-Pile
A full CRUD application built using the MERN stack.

# Installation and running the application
To run app: 
```git clone ```
```cd CRUD-pile```
```npm install```
```cd backend```
```npm install```
```npm run partytime```

Once your backend is successfully running, open a new terminal and then run:
```cd frontend```
```npm install```
```npm start```
